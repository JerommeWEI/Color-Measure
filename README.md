# Color-Measure

Color-Measure 是一个基于 Python `tkinter` 的光学与色度学参数计算工具。当前版本为 `v1.1`。

## 运行方式

```powershell
python optical_parameter_calculator.py
```

也可以在 Windows 下使用 `pythonw.exe` 启动，避免额外打开控制台窗口。

## v1.1 功能

### 物高计算

用于根据目标尺寸和光学参数估算工作距离、观察口径和物面分辨能力。

支持输入：

- 目标长度
- 目标宽度
- FOV 半视场角
- 入瞳直径
- 焦距

输出内容：

- 目标对角线
- 对角线半高
- 工作距离
- 2°观察角观测口径
- 中心物点可探测半锥角
- 中心物点可探测全锥角
- F/#
- 像面 Rayleigh 极限
- 物面 Rayleigh 极限
- 衍射极限等效 DPI
- 目标长度方向可分辨点数
- 目标宽度方向可分辨点数

### 色度学计算

用于评估光谱相机通道中心波长漂移对色度学结果的影响。

支持输入：

- 反射率 CSV 文件
- 采样起始波长
- 采样结束波长
- 采样间隔
- 单通道漂移量
- Lab 白点
- FWHM 模型
- 固定 FWHM
- FWHM 面阵标定 CSV
- FWHM 视场位置

计算逻辑：

- 以实测反射率曲线为输入
- 按设定波段和间隔进行标称采样
- 每次只模拟一个光谱通道发生正负漂移
- 启用 FWHM 时，按高斯光谱响应对反射率进行加权采样
- 导入面阵标定表时，过滤非 `ok` ROI 并按视场位置选择有效 FWHM
- 以中心 ROI 的 FWHM 为基准，计算中心到边缘 FWHM 变化造成的 Delta E00 偏差
- 重新计算 Lab
- 使用 CIEDE2000 计算漂移前后的 Delta E00

输出内容：

- 每个通道的平均 Delta E00
- 每个通道的最大 Delta E00
- 最大影响对应的测试样品
- 全局平均 Delta E00
- 全局最大 Delta E00
- 最敏感通道
- FWHM 场偏差的平均 Delta E00
- FWHM 场偏差的最大 Delta E00
- FWHM 场偏差 Top 5 ROI

### 界面

- 选项卡被选中后的视觉比例调整为未选中状态的 90%。

## 计算说明

本项目当前为轻量级单文件工具，不依赖第三方 Python 包。色度学计算内置 CIE 1931 2° 配色函数表和 CIEDE2000 计算公式。
